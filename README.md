# Employee Shift Scheduling System

## Description

A PostgreSQL database project for managing employee shifts, attendance, and payroll calculations.

## Tables

* Employees
* Shifts
* Shift_Assignments
* Attendance

## Features

* Shift assignment management
* Attendance tracking
* Payroll calculation
* Employee hour ranking
* Recursive weekly schedule generation
* Duplicate shift assignment prevention

## Technologies

* PostgreSQL
* pgAdmin 4
